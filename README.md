# 豆组筛选
豆瓣小组爬虫微信小程序版本
* 支持添加任意小组（一般就用在租房小组）
* 多关键词置顶、过滤
* 中介过滤 40%+

## 背景
* 豆瓣自带搜索功能太简单了
* 租房小组中介太多，没有自动过滤
* 其它工具会限定在某些小组里面，无法自定义
